# Boules Hisham Mokhtar — Portfolio

## About

This portfolio showcases featured projects, open-source packages, and professional experience. It auto-syncs public GitHub repositories via the GitHub API and includes hand-curated featured projects (including private/enterprise work).

## Tech

Built with plain HTML, CSS, and vanilla JavaScript — no frameworks, no build step.

- Responsive design with dark theme
- GitHub API integration (auto-fetches all public repos)
- Real-time language chart (Chart.js)
- Scroll-animated sections, filterable project grid, and contact form

## Deployment

Hosted on **GitHub Pages**. Any push to `main` triggers an automatic deploy via GitHub Actions.
