A personal portfolio website for Boules Hisham Mokhtar — Senior Flutter Engineer.

## Features

- Dynamic GitHub repository integration
- Language statistics visualization
- Responsive dark-mode design
- Interactive animations and filtering

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions on every push to the `main` branch. To enable deployment:

1. Go to the repository **Settings → Pages**
2. Under **Build and deployment**, set **Source** to **GitHub Actions**
3. The site will deploy automatically on the next push to `main`
